EXPERIMENTS

Clock Harmonizer アナログ時計の3本の針が、12音盤上で連続的なハーモニーを奏でる時計。
SYSTEM // ACTIVE FA-REPLAY // v1.0
? アナログ時計の文字盤に12音を割り当てています。時・分・秒の3本の針が時計として動きながら、リアルタイムにコード（和声）を生成し奏で続けます。
Digital Clock Synthesizer デジタル時計の「時:分:秒」の比率を純正律の周波数に変換するシンセ。
SYSTEM // ACTIVE FA-REPLAY // v1.0
? 刻一刻と変化するデジタル時計 of hh:mm:ss 表示を、そのまま整数比の周波数に落とし込み、純正律（Just Intonation）によるハーモニーを生み出します。逆再生（-100倍速）サウンドもおすすめです。
Circle of Fifths Visualizer アップロードした音声の周波数成分に応じて、五度圏の調性盤が光るビジュアライザ。
SYSTEM // ACTIVE FA-REPLAY // v1.0
? 音声ファイルを解析し、どのキー（音）が強く鳴っているかを五度圏（Circle of Fifths）のグラフィック上にリアルタイムでマッピングして光らせます。
ユーモア16タイプ診断 あなたの「笑いのツボ」を分析し、16タイプに分類する診断コンテンツ。
SYSTEM // ACTIVE FA-REPLAY // v1.0
